## 깃 관리

1. docs : 기획, 협업 관련 내용

3. back : 백엔드 프로젝트

## 컨셉

생성한 장르(카테고리)에서 원하는 친구와 관련 게시물을 공유하는 컨텐츠

## 특징

1. 사용자가 올리는 글은 비공개/장르에 생성된다
2. 장르는 사용자가 직접 생성한다
3. 장르는 장르 생성자, 초대를 수락한 사용자(친구)들만 볼 수 있다
4. 사용자는 글을 읽거나 작성/수정할 수 있고, 삭제는 장르 생성자와 글 생성자만 가능하다
5. 사용자는 글에 하트 버튼을 눌러 공감을 표현한다


## 페이지 구성
